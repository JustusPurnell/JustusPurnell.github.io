Project Overview

This website is being created for a local tattoo shop called Tattoo Supreme to enhance customer interactions. It will allow users to book appointments, view a gallery of tattoo work, and learn more about the shop's artists.

Intended Users

  • Potential customers looking to book a tattoo session.
  • Returning customers who want to interact with the shop.

Content Overview

  • Home Page: Introduction to the shop, showcasing the latest tattoo work.
  • Artists: Profiles of tattoo artists with their specialties and galleries.
  • Gallery: A portfolio of tattoos done by the shop.
  • Booking: Appointment form for users to book sessions.
  • Contact: Information about how to reach the shop.

Dynamic Features

  • A contact form that checks if the info entered (like name, email, and phone) is correct as you type.
